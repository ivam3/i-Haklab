# YouTubeDR

## ¿Qué es YouTubeDR?

**YouTubeDR** es una herramienta de línea de comandos para **descargar contenido de YouTube**, incluyendo videos, audio o listas de reproducción, generalmente basada en motores como `yt-dlp`.

## ¿Para qué es útil?

* Descarga de videos educativos
* Extracción de audio
* Automatización de descargas
* Uso offline de contenido

## Ejemplos de uso

**Descargar un video:**

```bash
youtubedr https://youtube.com/watch?v=XXXX
```

**Descargar solo audio:**

```bash
youtubedr --audio https://youtube.com/watch?v=XXXX
```

## Consideraciones legales

Respeta los términos de servicio de YouTube
Úsalo solo para contenido permitido
Evita redistribución no autorizada
